# 🗓️ Daily Logs

Append-only archive of concluded work, one file per day: `YYYY-MM-DD.md`.

When a conversation concludes (after commit), append its **compacted summary** here, then delete
the original `Conv-…` note. This is the long-term trail of what happened and why.

> ⚠️ Windows: write these with UTF-8 editing tools, not PowerShell `>>`/`Out-File` (UTF-16 corrupts the file).
