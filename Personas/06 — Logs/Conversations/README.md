# 🧵 Conversations

One note per work thread, to keep contexts segregated (Phase 2 of `AGENTS.md`).

- **Naming**: `Conv-YYYY-MM-DD-Brief-Topic.md` (e.g. `Conv-2026-06-17-Phase0-Scaffold.md`).
- **Header**: use the template in `AGENTS.md` → Phase 2 (front-matter + Objectives + Files Touched + Compacted Session Log + Memory Links).
- **On conclude**: after a successful commit, append the compacted summary into the day's
  [[06 — Logs/Daily Logs/|Daily Log]] and delete the conversation file.
